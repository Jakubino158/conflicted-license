["Commercial Use (Non-Competitive)","Describes the ability to use the licensed code in a commercial product or service that does not compete with the software"],
["Compensate for Damages","Describes the obligation to compensate the author for any damages cased by your work."],
["Contact Author","Describes your ability to contact the author about the module you're using."],

["Distribute","Describes the ability to distribute original or modified (derivative) works."],
["Give Credit","Describes the need to give explicit credit to the author when distributing the software."],
["Hold Liable","Describes the warranty and if the software/license owner can be charged for damages."],
["Include Copyright","Describes whether the original copyright must be retained."],
["Include Install Instructions","Describes the necessity to include build & install instructions."],

["Modify","Describes the ability to modify the software and create derivatives."],
["Pay Above Use Threshold","Described the obligation to pay the licensor after a certain amount of use."],
["Place Warranty","Describes the ability to place warranty on the software licensed."],
["Private Use","Describes the ability to use/modify software freely without distributing it."],

["Rename","Describes the ability to change software's name if modified/distributed."],
